# Online Calculator

A free, privacy-friendly online calculator with Normal and Scientific modes, a calculation history log, and full keyboard support. Hosted as a static site on GitHub Pages.

**Live site:** https://calculator.aysmartai.com/

## Features

- Standard arithmetic: addition, subtraction, multiplication, division, percentages, sign toggle
- Scientific mode: sin, cos, tan (degrees), √, x², x^y, log, ln, x!, 1/x, mod, π, e
- Calculation history log (click an entry to reuse its result, clear anytime)
- Full keyboard shortcut support
- No sign-up, no tracking, no server calls — everything runs client-side in the browser

## Tech

Single self-contained `index.html` file (HTML, CSS, vanilla JavaScript). No build step, no dependencies, no external requests. `robots.txt` and `sitemap.xml` are included for search engine indexing.
